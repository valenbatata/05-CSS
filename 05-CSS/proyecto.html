Trabajo de CSS

Background-color

Background-color es un propiedad de CSS que define el color de fondo de un elemento, puede ser el valor de un color o la palabra "transparente".

Background-image

La propiedad CSS background-image establece una o más imágenes de fondo para un elemento.

Color

La propiedad de CSS color selecciona el valor de color de primer plano del contenido de elemento de texto y decoraciones de texto. Tambien establece el valor currentcolor (en-US) que se puede usar como un valor indirecto en otras propiedades, y es el valor predeterminado para otras propiedades de color, como border-color.

Text-align

La propiedad CSS de alineación de texto establece la alineación horizontal del contenido de nivel en línea dentro de un elemento de bloque o cuadro de celda de tabla. Esto significa que funciona como alineación vertical pero en dirección horizontal.

width

La propiedad CSS ancho establece el ancho de un elemento. De forma predeterminada, establece el ancho del área de contenido, pero si el tamaño del cuadro se establece en border-box, establece el ancho del área del borde.

Heigh

La propiedad CSS de altura especifica la altura de un elemento. De forma predeterminada, la propiedad define la altura del área de contenido. Sin embargo, si el tamaño del cuadro se establece en cuadro de borde, en su lugar determina la altura del área del borde.

Border

La propiedad CSS abreviada de borde establece el borde de un elemento. Establece los valores de border-width, border-style y border-color.

Border-collapse

La propiedad CSS border-collapse establece si las celdas dentro de una tienen bordes compartidos o separados.

Display

La propiedad CSS de visualización establece si un elemento se trata como un bloque o un elemento en línea y el diseño utilizado para sus elementos secundarios, como el diseño de flujo, la cuadrícula o la flexión. la propiedad de visualización establece los tipos de visualización interior y exterior de un elemento. El tipo externo establece la participación de un elemento en el diseño de flujo
